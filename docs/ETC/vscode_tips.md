# VS Code Tips

## Change your theme

```
Command + K Command + T
```

### 현재 쓰고 있는 거

```
Material Icon Theme
```



## Shortcuts

### Terminal

```
Control + `
```

### Toggle Sidebar

```
command + b
```

### Zen mode(전체화면)

```
Command + k z
```



### Side by Side editing

```
Command + \
```

### Switch between editors

```
Command + 1, Command + 2 ...
```



## Editing

### Multi cursor selection (여러 위치 선택)

```
Option + Click
```

### 커서가 위치한 단어와 동일한 모든 단어 선택

```
Shift + Command + l
```

### 현재 위치에 선택한 단어와 동일한 단어 하나씩 추가

```
Command + d
```

### Column selection

```
Shift + Option + 마우스 드래그
```
