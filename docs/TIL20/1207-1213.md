# 1211

## 3장 언어의 기초

- `Number()` 와 `parseInt()` 는 단순히 상위호환의 관계일까?

  - 서로 역할이 있기때문에 둘다 남아있지 않을까?

  - `parseInt()`

    - "해석해서, Int형으로 뽑아낼꺼야."
      "문자열이 들어올거야. Number형으로 반환해낼거야."

    - "넘어온 값을 해석해서, 무언가를 만들어낼거야."

    - ```javascript
      parseInt("1234blue");	// 1234
      parseInt("blue1234");	// NaN
      ```

    - 다른 언어에서도 형변환 + 추출하는 메소드에는  `parse` 라는 접두어를 많이 쓴다고한다.
