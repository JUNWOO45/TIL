### Multiline ellipsis

- https://stackoverflow.com/questions/46152850/webkit-box-orient-styling-disappears-from-styling

```scss
overflow: hidden;
text-overflow: ellipsis;
display: -webkit-box;
-webkit-line-clamp: 2;
/* autoprefixer: off */
-webkit-box-orient: vertical;
/* autoprefixer: on */
```
