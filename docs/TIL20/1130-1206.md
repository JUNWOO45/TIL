## useMemo

성능 최적화를 위해 사용된다.

```typescript
...
const MyFunc = ({a, b}) => {
  const c = compute(a, b);
  
  return <div>{c}</div>
}
...
```

연산된 값을 useMemo hook을 사용하여 재사용할 수 있다.

a, b값이 바뀐 경우에만 compute함수를 호출하도록 변경해보자.

```typescript
...
const MyFunc = ({a, b}) => {
  const c = useMemo(compute(a, b), [a, b])
  
  return <div>{c}</div>
}
...
```
