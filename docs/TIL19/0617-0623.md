## 향상된 객체 리터럴

- 객체의 속성을 메서드로 할때, function 예약어를 생략하고 생성가능

  ```
  var dictionary = {
  	words: 100,
  	lookup: function() {
  		console.log('hello');
  	},
  	//lookup() {
  		console.log('hello');
  	}
  }
  ```

### Modules - 자바스크립트 모듈화 방법

- 자바스크립트 모듈 로더 라이브러리(AMD, Common JS) 기능을 js언어 자체에서 지원

- 호출되기 전까지는 코드 실행과 동작을 하지 않는 특징이 있습니다.

  ```
  // libs/math.js
  export function sum(x, y) {
  	return x + y;
  }
  
  // main.js
  import { sum } from 'libs/math.js';
  sum(1, 2);
  ```
