### string.split()

```
test_str = "web-js-python"
test_str.split('-')	//['web', 'js', 'python']

test2_str = "web js py"
test2_str.split(' ')	//['web', 'js', 'py']

test3_str = "bcabadefaghi"

print(test3_str.split('a'))
```

기본적으로 메서드 사용방법은 js와 동일한듯합니다!
