## deploy

###웹

```
$ gcloud app deploy -v <버전> --project="<프로젝트명>" --no-promote
```

버전 적어주고, dev인지 standby인지 프로젝트명도 적어주고, --no-promote로 트래픽이전 막고 배포.

### 앱

가장 중요한건, **build와 build-production** 을 잊지말것.

```
build.sh(build-prod.sh) 실행 후, static에서
$ gcloud app deploy -v <버전> --project="<프로젝트명>" --no-promote
```
